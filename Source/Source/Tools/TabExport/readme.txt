说明：
1. 名称：剑网3配置文件导出工具；
2. 行为：将保存在数据库(sql server)中的剑网3配置文件导出成tab文本文件，供剑网3客户端、服务端程		 序读取；
3. 目的：与每日构建配合使用，每日定时执行此工具导出配置文件，并提交配置文件至产品库，解决策划手		 工提交配置文件带来不便及易出错的问题；

4. 工具组成：a) 一配置文件TabExport.ini b)一可执行程序TabExport.exe c)一说明文档 readme.txt

5. 使用说明：
   step1: 详细阅读说明文件readme.txt
   step2: 配置TabExport.ini，配置说明见TabExport.ini
   Step3: 执行TabExport.exe
